## 面板说明

事件面板用于为选中组件绑定事件处理函数，实现组件的交互逻辑。支持配置组件支持的所有事件，如change、click、focus等。

## 功能特性

**事件列表**
- 展示方式: 列出组件支持的所有事件
- 事件类型: 不同组件支持不同的事件类型
- 例如: Input组件支持change、blur、focus等事件
- 例如: Select组件支持change、visible-change等事件

**事件编辑**
- 编辑方式: 通过FnEditor代码编辑器编辑事件处理函数
- 函数签名: 函数接收组件事件参数和excontext上下文对象
- 返回值: 根据事件类型决定是否需要返回值

**函数类型**
- COMPONENT_EVENT_FN: 组件事件函数类型
- 参数: (event: ComponentEvent, excontext: Recordable) => void
- 用途: 处理组件的原生事件

## 使用场景

- **数据联动** - 一个组件的事件触发另一个组件的更新
- **表单校验** - 在事件中执行自定义校验逻辑
- **数据提交** - 在事件中处理数据提交逻辑
- **UI交互** - 实现复杂的UI交互效果

## 注意事项

- 事件函数中可通过excontext访问上下文对象
- 事件函数应避免副作用，保持函数纯净
- 建议将复杂逻辑封装到上下文对象的方法中
