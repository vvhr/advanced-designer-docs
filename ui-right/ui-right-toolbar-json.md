# 右侧工具栏-代码

## 面板说明

代码面板以JSON格式展示和编辑选中组件的完整配置，包括所有属性、事件、插槽等配置项。适合熟悉JSON格式的高级用户使用。

## 功能特性

### JSON展示

格式: 完整的DesignerFormSchema对象
结构: 包含组件的所有配置信息
格式化: JSON自动格式化，便于阅读和编辑

### JSON编辑

编辑方式: 通过JSON编辑器直接编辑组件配置
语法高亮: 支持JSON语法高亮，提高编辑体验
错误提示: 编辑时进行JSON语法检查，发现错误及时提示

### 实时同步

双向同步: JSON编辑后立即同步到画布和属性面板
配置验证: 编辑时验证配置的有效性
错误恢复: 配置错误时提示并支持恢复

## 使用场景

批量配置: 通过编辑JSON快速批量修改组件配置
高级配置: 配置一些在属性面板中不直接支持的属性
配置调试: 调试组件配置问题时，直接编辑JSON进行测试
配置迁移: 复制组件配置JSON，在其他位置复用

## 注意事项

JSON编辑需要了解DesignerFormSchema的数据结构
错误的JSON格式会导致组件无法正常渲染
建议在编辑前先复制备份，避免误操作丢失配置
修改后注意检查画布中的组件是否正常显示
