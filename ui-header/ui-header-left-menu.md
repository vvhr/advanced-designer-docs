# 顶部左侧菜单栏功能

## 文件菜单

导入: 从JSON文件导入表单配置，支持覆盖当前设计或追加到现有设计
保存: 保存当前表单设计到父窗口（通过postMessage发送designer-save消息）
导出: 导出当前表单设计为JSON文件，供本地保存或分享

## 编辑菜单

清空: 清空当前画布中的所有组件，恢复为空白表单（需确认操作）

## 视图菜单

浅色主题: 切换到浅色主题模式
深色主题: 切换到深色主题模式
手机视图: 切换画布为手机视图模式（移动端预览）
平板视图: 切换画布为平板视图模式（平板预览）
笔记本视图: 切换画布为笔记本视图模式（小屏PC预览）
PC视图: 切换画布为PC视图模式（桌面端预览）

## 运行菜单

重置填写: 清空表单中所有已填写的值，恢复为初始状态
模拟提交校验: 触发表单校验，显示所有校验错误信息
真机模拟: 在新窗口中打开表单预览，模拟真实运行环境

## 版本菜单

自动保存设置: 配置自动保存的间隔时间和触发条件（功能开发中）
切换指定版本: 切换到历史版本，查看或恢复之前的表单设计（功能开发中）

## 帮助菜单

组件库文档: 打开advanced-ele-ui组件库的文档链接
设计器文档: 打开设计器使用文档（功能开发中）
AI智能助手: 打开AI助手浮动窗口，获取设计建议和帮助
清理缓存: 清理设计器的本地存储缓存数据
关于: 显示设计器版本信息和项目信息
