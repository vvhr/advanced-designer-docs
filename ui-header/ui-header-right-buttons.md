## 功能概述

顶部右侧提供四个快捷按钮，用于快速访问AI助手、查看日志、保存设计和关闭设计器。

## AI客服按钮

- **功能** - 打开AI助手浮动窗口
- **用途** - 提供智能表单设计建议、组件使用指导、问题解答等AI辅助功能
- **显示** - 点击后弹出浮动窗口，可随时关闭

## 日志按钮

- **功能** - 打开内部日志查看器
- **用途** - 查看设计器运行过程中的日志信息，包括错误、警告和调试信息
- **显示** - 日志数量超过0时，按钮上会显示未读数量徽章（最多显示99+）
- **交互** - 点击后弹出日志查看器对话框，可查看、筛选和清空日志

## 保存按钮

- **功能** - 保存当前表单设计
- **用途** - 将当前设计的表单配置发送给父窗口进行持久化保存
- **流程** - 点击后触发保存流程，通过postMessage发送designer-save消息，等待父窗口响应（最多等待10秒）
- **反馈** - 根据父窗口返回的designer-save-success或designer-save-failed消息显示保存结果

## 关闭按钮

- **功能** - 关闭设计器
- **用途** - 退出设计器，通知父窗口关闭iframe或弹窗
- **交互** - 点击后通过postMessage发送designer-close消息给父窗口，由父窗口决定如何处理关闭操作
