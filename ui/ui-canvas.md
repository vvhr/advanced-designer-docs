# 中间设计器画布

## 面板说明

画布是表单设计的核心区域，实时预览和编辑表单。支持拖拽排序、组件选择、视图模式切换等功能。

## 功能特性

### 表单预览

实时渲染: 表单配置修改后立即在画布中渲染
交互支持: 支持表单的正常交互，如输入、选择、提交等
样式预览: 实时预览表单的样式和布局效果

### 视图模式

PC视图: 桌面端视图模式，默认宽度
手机视图: 移动端视图模式，窄屏显示
平板视图: 平板视图模式，中等宽度
笔记本视图: 小屏PC视图模式

### 设计模式

组件边框: 设计模式下显示组件的边框和操作手柄
拖拽手柄: 显示拖拽排序的手柄，支持拖拽调整顺序
操作按钮: 显示组件的操作按钮（复制、删除、锁定等）

### 组件操作

选择组件: 点击组件可选中，右侧面板显示配置项
拖拽排序: 通过拖拽手柄调整组件顺序
复制删除: 通过操作按钮复制或删除组件
锁定解锁: 锁定组件后不可编辑和删除

## 交互特性

### 拖拽添加

从组件面板拖拽组件到画布指定位置
自动插入到目标位置，保持表单结构

### 拖拽排序

在画布中拖拽组件调整顺序
支持在同一层级内调整，也支持跨层级调整

### 组件选择

点击组件选中，高亮显示
选中后右侧面板自动显示该组件的配置项
支持通过大纲面板快速定位和选择组件

### 实时预览

配置修改后立即在画布中反映
支持实时查看表单在不同数据值下的效果

## 使用场景

表单设计: 通过拖拽和配置快速构建表单
实时预览: 实时查看表单的展示效果
交互测试: 在画布中测试表单的交互行为
样式调整: 调整表单的布局和样式
