# 左侧工具栏-组件

## 面板说明

组件面板提供设计器可用的所有组件，支持通过拖拽方式将组件添加到画布中。组件按类型分类展示，包括基础组件和示例组件。

## 组件分类

### 装饰型组件（Decorator）

用于展示和装饰的组件，不参与表单数据收集。包括Alert、Text、Image等组件。

### 输入型组件（Inputer）

用于数据输入的组件，可收集用户输入的数据。包括Input、Select、DatePicker、Upload等组件。

### 容器型组件（Container）

用于布局和分组的组件，可包含子组件。包括Group、Step等组件。

### 描述列表组件（Descriptions）

用于数据展示的组件，以描述列表形式展示数据。

## 组件操作

### 添加组件

方式: 从组件面板拖拽组件到画布指定位置
效果: 组件添加到画布后自动选中，右侧面板显示该组件的配置项

### 组件搜索

功能: 在组件面板顶部搜索框输入关键词，快速定位组件
支持: 按组件名称、描述、类型等关键词搜索

### 组件预览

功能: 鼠标悬停在组件上可查看组件描述和图标
说明: 每个组件都有对应的图标和简短描述，帮助用户选择合适的组件

## 组件配置

组件添加到画布后，可通过右侧属性面板进行详细配置，包括基本属性、组件属性、表单项属性、布局属性等。
