# 左侧工具栏-上下文

## 面板说明

上下文面板用于配置表单的外部上下文对象，为组件提供动态数据和方法。上下文对象可在表单Schema函数中使用，实现动态配置。

## 功能特性

### 上下文编辑

编辑方式: 通过JSON编辑器编辑上下文对象
数据类型: 支持对象、数组、字符串、数字等JSON支持的所有数据类型
实时生效: 上下文修改后立即生效，影响所有使用该上下文的组件

### 上下文使用

在Schema函数中使用: 通过excontext参数访问上下文对象
在事件函数中使用: 组件事件处理函数中可通过excontext访问上下文
动态数据绑定: 使用上下文中的数据实现动态属性值

## 使用场景

API数据注入: 将API返回的数据注入到上下文中，供组件使用
方法注入: 将业务方法注入到上下文中，供组件事件调用
全局配置: 将全局配置信息放入上下文，供所有组件共享

## 注意事项

上下文对象会被序列化，仅支持可序列化的数据类型
上下文修改会影响所有使用该上下文的组件，需谨慎操作
建议将上下文对象设计为扁平结构，便于查找和使用
