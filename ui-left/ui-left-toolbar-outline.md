# 左侧工具栏-大纲

## 面板说明

大纲面板以树形结构展示表单中所有组件的层级关系，便于快速定位和调整组件顺序。

## 功能特性

### 树形展示

结构: 按照组件在表单中的嵌套关系展示为树形结构
层级: 支持多级嵌套，容器组件下可包含子组件
标识: 每个节点显示组件类型、字段名、标签等关键信息

### 快速定位

点击节点: 点击大纲中的组件节点，画布中对应组件自动选中并滚动到可视区域
高亮显示: 选中组件在大纲中高亮显示，便于识别当前操作对象

### 拖拽排序

支持拖拽: 在大纲中可通过拖拽调整组件顺序
实时同步: 拖拽调整后，画布中的组件顺序同步更新

### 组件操作

右键菜单: 支持右键菜单进行复制、删除、锁定等操作
批量操作: 支持选中多个组件进行批量操作

## 使用场景

复杂表单设计: 当表单组件较多时，通过大纲快速定位和导航
层级调整: 调整组件嵌套关系和顺序
组件查找: 通过大纲快速查找特定组件
