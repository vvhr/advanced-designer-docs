## 面板说明

数据面板用于查看和编辑表单数据模型，实时预览表单值的变化。面板显示当前表单中所有字段的值，支持直接编辑。

## 功能特性

**数据展示**
- 格式: 以JSON格式展示表单数据模型
- 结构: 按照表单字段的field属性组织数据
- 实时更新: 画布中表单值变化时，数据面板自动同步更新

**数据编辑**
- 编辑方式: 通过JSON编辑器直接编辑数据值
- 类型支持: 支持字符串、数字、布尔值、数组、对象等数据类型
- 立即生效: 编辑后立即反映到画布表单中

**数据验证**
- 类型检查: 编辑时进行数据类型验证
- 格式验证: 对JSON格式进行验证，确保数据有效性

## 使用场景

- **数据预览** - 查看表单在不同数据值下的展示效果
- **数据测试** - 快速设置测试数据，验证表单行为
- **数据调试** - 调试表单逻辑时，快速修改数据值进行测试

## 注意事项

- 数据面板显示的是表单的model值，不是组件的value值
- 修改数据面板的值会影响画布中表单的实际值
- 建议在测试和调试时使用，生产环境不建议直接编辑
